# Team21LargeProject
The GitHub for Team 21's Large Project for COP 4331 Spring 2026.
ParcelPantry: The Household Logistics Engine 
Overview: Parcel Pantry is a web application designed as a Household Logistics Engine and implemented using a MERN-stack. Its primary purpose is to streamline communal living by providing a centralized hub for managing shared groceries, tracking household inventory, and facilitating communication between roommates or family members. Key features include account creation using email verification, as well as House creation that will connect users to one another.
